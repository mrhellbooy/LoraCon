# LoraCon: Project Roadmap & Planning

## 🎯 Vision
To build the most advanced, decentralized, and user-friendly VPN orchestration ecosystem that bridges the gap between bio-organic design and military-grade security.

- [ ] **Android Production Candidate**: Finalize the Kotlin-based `VpnService` with WireGuard integration.
- [ ] **Solana Mainnet Migration**: Transition from Devnet to Mainnet for real value exchange.
- [ ] **Sentinel Mesh**: Deploy 20 additional high-performance nodes in privacy-friendly jurisdictions.
